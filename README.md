# Blog Stats SVG Generator

Automatically generates an SVG visualization of your blog post history for embedding in GitHub profiles or READMEs.

## Setup

1. Fork this repository
2. Add your `BLOG_URL` as a repository secret in your fork's settings
3. Enable GitHub Actions for your fork
4. The SVG will be automatically generated daily and committed to the `output` directory

## Manual Generation

You can manually trigger the generation workflow:

1. Go to the Actions tab
2. Select "Generate Blog Stats SVG"
3. Click "Run workflow"

## Embedding

Add this to your GitHub profile README or any other markdown file:


# Latest run blog history

![image](./output/blog-history.svg)
